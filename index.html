Voyeur by devthenet

This is a rubygem that converts almost any video to HTML5 and other formats. It's designed from the ground up to be easy to use, we've tried other tools and they were a pain in the butt, not the kind of butt you want voyeur to convert to internet viewing goodness.

Video Codecs

We support any format that ffmpeg supports, if you can't seem to convert a video please contact us (or find a fix yourself and send a pull request) and we'll try to get it sorted for you.

We support three different output formats at the moment: .ogv, .webm and .mp4. If you want more, send a pull request with your tested code and we'll add it.

Cool, how do I use it?

Go to the github page and follow the instructions, boss. We tried to make it easy to install and use, if you find anything that isn't up to your high standards then you know what to do ;).
